# 雪峰山智慧文旅系统

#### Description
本仓库为怀化雪峰山五大景区定制开发的智慧文旅预约导览系统。采用 SpringBoot + Vue3 + UniApp 三端架构，包含分时限流预约、AI智能问答、多景区核销与可视化数据大屏等功能。
